<h1> Architecture of our App</h1>
<h2> Component Architecture</h2>
<h3> Pages:</h3> <p> Login.jsx, Home.jsx, To-Do.jsx, Settings.jsx, Profile.jsx, Mental.jsx, App.jsx</p>
<h3> Components:</h3> <p> Nav.jsx, Button.jsx, Header.jsx, Card.jsx</p>
<h3> Feature Components: </h3> <p>Activity.jsx, Distractions.jsx, Planner.jsx, Pop.jsx </p>
<h2> Backend Architecture: </h2> <p>Created a generalized database engineered to store all inputs made by users, including the login details. To handle the OAuth inputs regarding the "Sign in with Google" button, we used the stanrdized Google Authentication model created specifically to handle signin requests. For access on the cloud, all databases are being hosted on Firebase by Google.</p>
